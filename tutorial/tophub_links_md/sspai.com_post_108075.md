Chops：AI Agents 的本地技能管理工具

平台：macOS

关键词：AI、Agent、Skill

@Vanilla

：随着 AI 编程工具的普及，Skills 已经成为定制和扩展 Agent 行为的核心机制。但问题在于，每个工具都有自己的本地存储路径，散落各处的 Skills 文件难以形成统一管理，更谈不上跨工具检索。Chops 的出现正是为了解决这一痛点，它支持 Claude Code、Cursor、Codex、Windsurf、Copilot、Aider、Amp 等工具，可以实现一站式 Skills 管理。

首先，Chops 内置了一个编辑器，可以对 Skills 文件直接进行查看、编辑、保存等操作，免去了跳转到第三方文本编辑器的繁琐。除了 Markdown 格式的文件，Chops 还可以自动解析 YAML Frontmatter（元数据）和 Cursor 特有的

.mdc

文件格式。值得一提的是，Chops 还支持新建 Skills，并会自动生成对应工具的规范模板（boilerplate）。

其次，Chops 支持跨工具的整理、搜索和同步功能。Chops 可以自动识别本机上的 Skills，并按照不同的工具分类显示。对这些 AI Agents 中散落的 Skills，我们可以通过 Chops 中的喜爱和合集功能进行整理，帮助我们在需要时快速调用。如果你懒得整理也没有关系，Chops 支持对 Skills 文件的全文搜索，包括名称、描述和内容。Chops 对 Skills 文件的索引是基于 macOS 的 FSEvents 技术，当我在终端或其它编辑器修改了 Skills 文件，Chops 会实时刷新显示。

最后，Chops 还支持对远程服务器中 Skills 的管理。以 OpenClaw 为例，我们可以通过服务器信息和密钥来建立连接，然后让 Chops 远程管理 OpenClaw 中的 Skills，包括搜索、编辑、删除、安装等操作都不在话下。

如果你只想要一个纯粹的 Skills 管理工具，可以说 Chops 的功能已经相当完善，对于日常与多个 AI 编程工具打交道的开发者来说，Chops 是一个值得纳入工作流的效率工具。Chops 目前免费开源，你可以在

官网

直接下载安装，也可以从 GitHub 平台下载源码自行编译安装，不过目前仅支持 macOS 平台（需要 macOS 15 Sequoia 或更高版本）。

PaperKnife+：开源且全能的本地 PDF 工具箱

平台：Android

关键词：PDF

@大大大K

：现在手机系统大都支持拍照直接生成 PDF 扫描件，我们也越来越习惯这样操作，而不是再去找扫描仪了。但这种方便往往只停留在第一步，如果扫描之后发现还需要单独调整某个页面，或者要把多个 PDF 合并、拆分时，身边要是没有电脑就很难办了。如果此时我们手机里有这样一个能简单操作 PDF 文件的 App，那就方便很多了。

PaperKnife+ 就是这样一款 PDF 工具。它的 UI 设计非常出色，界面中的各种阴影、图标等细节非常精致，完全没有传统「工具型」App 那样沉重。功能排布采用非常直观的卡片式设计，清晰又简洁。虽然 App 不支持中文显示，但也都是非常简单的英文描述，大概一看就能看懂。体验下来非常轻盈丝滑，质感极佳。
