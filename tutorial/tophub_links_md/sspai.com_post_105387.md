NotePlan 3.2：支持简体中文、新增集成式 HTML 插件等

平台：iOS / iPadOS / macOS / Web

关键词：任务看板、团队协作

@ElijahLee

：集任务、笔记、日历三合一的时间管理工具 NotePlan 更新了 3.2 版本，应用新增多语言支持，现已支持简体中文，另外还有用于快速记录的锁屏小组件、集成式 HTML 插件、Liquid Glass 设计风格支持更新等。

首先是支持多语言支持。 NotePlan 3.2 版全面支持简体中文，对于这种复杂又专业性较强的应用来说，本地语言支持是非常有助于用户快速上手的。另外，新版加入了对日语、意大利语、韩语、俄语的本地化，以及支持 VoiceOver 无障碍。

在移动端，新版更新增加了一个专注于快速捕捉的锁屏小组件。你可以直接从锁屏界面添加任务、启动语音转写，或创建一条笔记内容。

这个锁屏小组件支持自定义配置，你可以在小组件设置中配置笔记内容的保存位置，以及在哪个标题下；在内容方面，可以通过编辑器使用丰富的富文本格式设置，包括任务、项目符号、引用等；在功能方面，支持语音转写、键入文字、手写、绘图等多种方式。

在 macOS 端，NotePlan 推出了集成式 HTML 插件，并且提供了多个插件模板和

教程

帮助用户快速上手。在应用的「设置 - AI 与插件」中，我们可以对所有插件进行管理、下载更新并使用。

本次更新的集成式 HTML 插件主要是

Dashboard 插件

和 Linear Calendar 插件。 Dashboard 插件是一个仪表盘插件，可在侧边栏中显示统计、汇总视图、任务进度等，可用于快速查看工作概况。Linear Calendar（日历视图插件）提供增强的日历视图，集成了 macOS 日历和 NotePlan 日历于一体。

HTML 插件可以直接运行在 NotePlan 主窗口内。当你打开一个 HTML 插件后，它会自动出现在侧边栏中，你可以像在笔记、日历或任务之间切换一样，随时切换到这些插件。这可以让基于 HTML 的插件在日常工作流中变得更加实用，而不再需要单独打开一个窗口。

NotePlan 官方提到其它更多插件正在开发中，但我们也可以跟着教程使用 Claude Code 或 Cursor 快速编写插件，即使不会写代码也可以上手。

在 NotePlan 3.19 中，应用支持了 Liquid Glass 设计风格。侧边栏变为悬浮样式并会穿透显示笔记背景颜色，命令栏与菜单更新为更透明的玻璃质感，全局图标更统一现代。
