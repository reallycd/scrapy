如果 AI 不再只是回答问题，而是开始悄悄改写价格、入口和责任边界，普通用户最先会在哪里感觉到变化？我会先看三处：OpenAI 把 Token 价格往下压，说明模型能力正在从稀缺品变成流量生意；豆包 Pro 和千问高考助手，把国产 AI 推进更具体的教育、创作与生活决策；Meta 与 Tesla Robotaxi 又提醒我们，平台规则和线下出行也在被重新接线。真正值得琢磨的不是哪家公司更会造概念，而是谁能把便宜、可用、可信这三件事同时交到用户手里。

一起来看看这两天发生的AI和互联网赛道的大事件吧！

AI赛道观察

1. OpenAI 拟大幅下调 Token 价格，模型价格战逼近 Anthropic

核心内容

：OpenAI 正考虑大幅下调 API Token 价格，以更激进的方式争夺开发者和企业客户。价格调整会直接影响应用厂商的调用成本，也会把 Anthropic、Google 等前沿模型公司的定价压力推高，模型能力竞争正在更快转向规模化使用竞争。

点评

：大模型降价不是单纯让利，而是入口争夺。谁先把调用成本压下来，谁就更容易成为开发者默认底座。

2. OpenAI 筹备超 10GW 算力联盟，星际之门继续扩张

核心内容

：OpenAI 继续围绕超大规模数据中心和能源供应推进算力联盟，目标把星际之门相关基础设施扩展到 10GW 以上。训练、推理、企业服务和智能体产品都需要长期算力支撑，前沿模型公司的竞争越来越像基础设施战争。

点评

：模型公司一边降价，一边锁算力，本质是用规模摊薄成本。AI 价格战背后，拼的是电力、芯片和资本耐心。

3. Anthropic 发布 Claude Fable 5 数据留存新规，企业客户成豁免重点

核心内容

：Anthropic 调整 Claude Fable 5 的数据留存规则，普通用户和部分付费用户会被纳入新的数据保存与改进模型流程，企业和 API 客户则获得更明确的豁免安排。前沿模型公司正在平衡模型迭代、隐私承诺和商业客户信任。

点评

：数据留存会成为 AI 订阅的重要分层。个人用户看功能，企业用户更看控制权，厂商必须把边界说清楚。

4. Anthropic 安全测试曝光生物问答红线，Claude Fable 5 拒答率大幅提升

核心内容

：Claude Fable 5 在生物安全相关测试中显著提高拒答率，对可能涉及危险实验、病原体和滥用风险的问题采取更谨慎策略。随着模型推理能力增强，安全团队开始把重点放到高风险专业知识的访问控制和误用防范上。

点评

：越强的模型越不能只谈能力上限。安全拒答看起来保守，但它决定 AI 能否进入医疗、科研和政企场景。
