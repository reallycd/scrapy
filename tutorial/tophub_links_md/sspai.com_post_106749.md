Snapseed：新增内置相机和复古胶片滤镜

平台：iOS

关键词：手机摄影、复古胶片滤镜

@ElijahLee

：自从 Snapseed 的 iOS 版本进入 3.0 版本以来，一直积极保持更新，近期应用推出了 3.13、3.15 迭代与更新，新增 Snapseed 相机、复古胶片滤镜、锁屏小组件等功能。

首先是应用在首页新增一枚相机按钮，它出现在应用的右上角。目前 Snapseed 相机主要围绕胶片滤镜效果，目前提供十余款免费的胶片滤镜，包括以 Kodak、Fujifilm 等特定胶片为灵感设计的模拟滤镜，主要风格为复古氛围。

在拍摄时，Snapseed 相机提供了自动模式和手动模式可选，使用取景器下方的拨轮即可切换手动模式，可以调节的控制功能包括 ISO、快门速度和对焦，以及闪光灯和变焦功能。手动调节后，取景器中的参数值与画面效果会实时调整，所见即所得。如果想回到自动模式，点击参数方框即可恢复自动。

Snapseed 相机的推出意味着 Snapseed 不再只是单纯的修图软件，而是开始向轻量级拍摄工具转变。

使用 Snapseed 相机拍摄的照片，保留了完整的原图和编辑图层，这意味着我们还可以在应用中继续对照片修饰调整，并且支持保存到原图。

在 3.13 版本中增加的胶片颗粒样式，可以为照片增加颗粒效果，还原摄影作品中的复古感觉。在编辑页面顶部进入粗粒胶片编辑页面，在画面上下滑动即可选择颗粒参数，接下来在下方拨轮左右滑动即可调整数值。

在其他方面，Snapseed 为 iPhone 增加了锁屏小组件支持，在锁屏页面即可启动 Snapseed 相机进行拍照，另外也支持相机控制按钮启动 Snapseed 相机。在工具方面，全新的去雾功能可以通过分析照片的雾霾和阴影，通过增加对比度、锐化等效果，实现去除大气污染的效果；升级的颜色工具，可以分析图片中相似色调的区域，对这些局部区域进行快速的色彩调整。

CoPaw：低门槛的「国产版 OpenClaw」

平台：Windows / macOS / Linux / Web

关键词：AI、AI 助手

@大大大K

：要说最近 AI 圈什么最火，那非 OpenClaw 莫属，不过它本身使用门槛不低，许多配置项还需要通过 json 文件修改，而且它对 Windows 环境支持并不尽如人意。不过近期，阿里旗下的团队开源了一款类似工具名为 CoPaw，原生的中文语言支持和更贴近国人使用环境的图形化配置界面，极大程度上降低了使用门槛。
