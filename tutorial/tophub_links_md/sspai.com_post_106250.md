Cubox：全新的桌面 Web 版

平台：Web

关键词： 稍后阅读、AI 辅助

@ElijahLee

：稍后阅读 + 知识管理应用 Cubox 推出了全新的桌面 Web 版，更新全新的视觉界面与多种布局、功能强大的详情控制面板、支持 AI 辅助阅读等功能。

首先， 桌面 Web 版采用多栏式的分布，按照软件常用的布局排版，从左至右分别是文章分类、文章卡片、阅读区以及最右的信息栏。文章卡片的布局也有多种样式可选：默认的单列双栏，按照卡片 + 阅读两栏布局。多列平铺，将所有文章以卡片形式平铺在屏幕上，点击才能进入阅读，适合那些仅通过标题即可筛选文章的用户。

阅读区左上角的扩展按钮，可以折叠所有功能区，仅保留阅读器，进入全屏沉浸式阅读。这时，使用 Comnand + 方向键，可以快速切换上下篇，不需要回到多栏布局，保留沉浸式阅读体验的延续性。

Cubox 接入 AI 辅助阅读， 打开右侧信息栏，这里提供了基于 AI 梳理的文章快速解读、关键问题和快速高亮等。问答功能， 基于 AI 对话的提问，我们可以键入与文章有关的问题或想了解的知识，快速获得答案；应用也预设大量常用问题，如简单总结、文章的关键论点、写一段分享文案等常用指令。

信息栏最右侧的信息控制面板，可以对文章的所有信息自定义编辑，包含文章标题、简介、设置分类、添加标题。遇到感兴趣的文章时，我们可以在这里随时编辑整理，加入收藏或分类标记，快速保存重要资料，方便日后查阅或回顾。

批量编辑功能也是桌面 Web 版的特色亮点，当光标移动到文章卡片右上角的勾选框位置后，选框会自动放大，点击即可进入批量选择模式，快速勾选多篇文章，并在右侧，编辑所有信息、增加标签、设置星标等等。在多列平铺布局时，批量编辑效率进一步强化，所有操作会排列在顶部，将多个内容拖动到目标功能即可完成操作，非常高效。

针对用户呼声较高的功能，桌面 Web 版也有许多改进，包括：收集箱 Inbox 的文字默认为未读， 完成阅读后自动消失；标注列表支持单列和多列显示方式、批量操作。搜索功能保留快速查找和详情搜索，前者帮助快速定位内容，后者用于批量浏览或管理一组内容。

总之，Cubox 的桌面 Web 版针对网页端性能，支持了沉浸式阅读、批量整理、精细标注、AI 辅助、实时同步的功能，同时搁置了一些非核心功能，适合高密度长文阅读场景，也适合后期系统整理与反复调用。你可以通过
