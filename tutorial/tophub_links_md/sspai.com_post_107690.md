Beam Browser：为 iPad 打造的专属浏览器

平台：iPadOS

关键词：浏览器

@ElijahLee

：Beam Browser 是一款针对 iPadOS 开发的全新浏览器应用，应用主打将 iPad 浏览器变成生产力工具，设计侧边栏、工作空间、支持 AI 等功能，并且将 iPad 的触控操作、键盘生态融入到浏览器的功能中。

在 Beam Browser 中，浏览的网页在侧边栏进行纵向排列，相较于 Safari 的顶部排列方式，侧边栏更符合大尺寸横向屏幕，并且可以同时看到多个标签标题，比传统横向标签高效很多。Beam Browser 侧边栏支持自动隐藏，给使用带来沉浸式的体验，用手指在屏幕上滑动即可将它呼出，非常方便。在功能上，它还支持固定网页标签、拖拽排序、文件夹嵌套等高级功能。

Beam Browser 在侧边栏还提供了 Space（工作空间）功能，这一点与 Arc 有点类似。我们可以为工作、生活等特定场景频繁浏览的网页，设置工作空间。每个工作空间都是一个独立的环境，拥有相互独立的标签、历史记录和缓存等内容。应用为工作空间提供了大量的图标 icon，并且支持自定义颜色来加强区分辨识。

Beam Browser 在应用内设计了 Command Bar（命令栏）功能，使用快捷键

Command - T

打开，它类似于 Spotlight 可以直接搜索网页、切换空间、使用 AI 或者执行特定的操作。这离不开应用对键盘和快捷键的支持，Beam Browser 的快捷键与 macOS 体系基本一致，打开/关闭网页、查找、重新加载等都与 Safari 无异。应用也有新的指令，使用 @ 符号可以选择在 Google、DuckDuckGo 等搜索引擎快速搜索。

Beam Intelligence 使用

Command - J

即可调用。它基于 Apple 智能，可以通过 AI 与网页对话，提供了包含总结网页内容、分析核心观点、简要解释内容、提取信息等等分析总结。该功能也支持用户自行切换到第三方大模型，包括 Claude、Google、OpenAI 等。

此外应用还有内置广告拦截、本地存储数据、网页悬浮预览、iCloud 同步、强制深色模式、阅读模式、网页翻译等实用功能。总的来说，Beam Browser 为 iPad 打造了一款实用又好用的专属浏览器。你可以在

App Store

下载，应用售价 28 元。
